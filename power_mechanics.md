# POWER MECHANICS — MERGE EDITION

This game uses three power mechanics only.

---

# 1️⃣ DIRECT DEATH CHALLENGE (Pre-Merge)

Use once before merge.

Example: Silent Line Challenge.

Players line up based on criteria.
If incorrect:
- Two worst positions are instantly banished.
- No debate.
- No vote.

Purpose:
Accelerates early elimination.

---

# 2️⃣ THE DAGGER (Merge Challenge Reward)

Earned via merge challenge.

Power:
- Cast TWO votes in one banishment.

Rules:
- May use this round or next.
- Must declare before vote reveal.
- Votes must go to same player.
- Expires after 2 rounds.
- Cannot use at Final 6.
- If holder is banished → power disappears.

Purpose:
Creates political leverage.

---

# 3️⃣ THE CHALICE MARK (One Round Only)

Activated during first merge round.

## Public Instruction

“Grab a chalice. We’re doing a ceremonial toast.”

3–4 minutes of mingling.

## Secret Rule (Traitors Only)

One Traitor may:
- Interlock arms with a player.
- Say “Cheers.”
- Sip together.
- Host must see it.

That player becomes MARKED.

## Effect

At night:
- Marked player dies automatically.
- Replaces normal murder.

Rules:
- Only usable once.
- Cannot use below 12 players.
- If no mark occurs → normal kill proceeds.

Purpose:
Creates undetectable in-plain-sight elimination.
